# Option-pricing-Dashboard-using-BS-model

Built an end-to-end derivatives risk analytics project using real NSE market data to value European options and assess market risk exposures.
Calculated historical volatility from stock return distributions and implemented Black-Scholes pricing for option valuation and fair value estimation.
Conducted sensitivity analysis (Delta/Vega), stress testing, and scenario analysis to measure option exposure under changing market conditions.
Built a risk dashboard summarizing valuation, volatility metrics, Greeks, intrinsic/time value, and model validation results.
Validated theoretical prices against traded NSE option premiums, achieving pricing error below 2% for selected contracts.
Observed that deep in-the-money, near-expiry options were primarily driven by intrinsic value, resulting in limited sensitivity to volatility assumptions and close alignment with market prices.
